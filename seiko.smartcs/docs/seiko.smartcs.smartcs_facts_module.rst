***************************
seiko.smartcs.smartcs_facts
***************************

**Collect facts from SmartCS**


Version added: 1.3.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Collects a base set of device facts from a SmartCS. This module prepends all of the base network fact keys with ``ansible_net_<fact>``.  The facts module will always collect a base set of facts from the device and can enable or disable collection of additional facts.

Notes
-----

.. note::
   - Tested against SmartCS NS-2250 System Software Ver 2.1
   - For more information on using Ansible to manage network devices see the :ref:`Ansible Network Guide <network_guide>`



Examples
--------

.. code-block:: yaml

    - name: Collect all facts from the device
      seiko.smartcs.smartcs_facts:
        gather_subset: all

    - name: Collect only the config and default facts
      seiko.smartcs.smartcs_facts:
        gather_subset:
          - config

    - name: Do not collect tty facts
      seiko.smartcs.smartcs_facts:
        gather_subset:
          - "!tty"
